# Установка и подготовка GitHub

## Требуемое ПО

### Для конечных пользователей

- Docker Desktop на Windows
- Docker Engine + Docker Compose на Linux

### Для контрибьюторов

- Python 3.10+
- Node.js 18+
- Docker

## Стандартная процедура запуска

1. Клонируйте репозиторий
2. Создайте `.env` на основе `.env.example`
3. Выполните `docker compose up --build`
4. Откройте `http://127.0.0.1:3000`

## Запуск без вспомогательных скриптов

### Linux

```bash
cp .env.example .env
docker compose up --build
```

### Windows

```bat
copy .env.example .env
docker compose up --build
```

## Артефакты GitHub Releases

Папка `release/` содержит готовые скрипты запуска:

- `release/linux/start-ics-risk.sh`
- `release/linux/stop-ics-risk.sh`
- `release/windows/start-ics-risk.bat`
- `release/windows/stop-ics-risk.bat`

Эти файлы предназначены для прикрепления к релизам вместе с архивом исходного кода.
